# 🧠 Models Covered in This Project

# 🎨 Variational Autoencoder (VAE)

A Variational Autoencoder is a generative model that learns latent representations of data.

It consists of:

✅ Encoder Network
✅ Latent Space Representation
✅ Decoder Network

VAE learns compressed representations and generates new data samples.

---

## 🔥 Key Features of VAE

* 📦 Latent Space Learning
* 🧠 Feature Extraction
* 🎨 Data Generation
* 🔍 Dimensionality Reduction
* 🤖 Representation Learning

---

## 🌍 Applications of VAE

* 🖼 Image Generation
* 🧬 Data Compression
* 📊 Anomaly Detection
* 🎨 Creative AI Systems
* 🤖 Generative Modeling

---

# ⚡ Generative Adversarial Network (GAN)

GANs are one of the most revolutionary models in Generative AI.

GAN consists of two neural networks:

## 🔹 Generator

Generates fake samples trying to imitate real data.

## 🔹 Discriminator

Distinguishes between real and fake samples.

Both networks compete against each other during training.

---

## 🔥 Key Features of GAN

* 🎨 Realistic Image Generation
* 🤖 Adversarial Learning
* 🧠 Data Synthesis
* 📸 High-Quality Outputs
* 🚀 Creative AI Applications

---

## 🌍 Applications of GAN

* 🖼 AI Image Generation
* 🎥 Deepfake Technology
* 🎮 Game Asset Creation
* 📷 Super Resolution
* 🎨 AI Art Generation

---

# 🌐 Graph Convolutional Network (GCN)

GCNs are neural networks specifically designed for graph-structured data.

Unlike traditional neural networks, GCNs can learn from:

✅ Nodes
✅ Edges
✅ Graph Relationships

GCNs are widely used in modern Graph AI systems.

---

## 🔥 Key Features of GCN

* 🌐 Graph Representation Learning
* 🔗 Relationship Understanding
* 🧠 Node Classification
* 📊 Structural Learning
* 🤖 Graph Intelligence

---

## 🌍 Applications of GCN

* 👥 Social Networks
* 🧬 Molecular Analysis
* 📈 Recommendation Systems
* 🚦 Traffic Prediction
* 🔍 Fraud Detection
